# Changelog

All notable changes to the "Gemini Code Assist RTL Support" extension will be documented in this file.

## [0.1.0] - Initial Release
- Adds an RTL (Right-to-Left) toggle button to the Gemini Code Assist chat interface.
- Automatically detects RTL languages (Hebrew, Arabic, Persian) on a per-bubble basis.
- Fixes alignment issues for RTL speakers while ensuring all inline/block code remains Left-to-Right.
- Includes a Status Bar button for quick activation and monitoring of the RTL patch status.
- Support for injecting the RTL logic into the Plan Preview feature.
